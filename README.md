# Pianorad 1926 – Fitch Edition (Cmajor Patch)

Emulation of the 1926 Hugo Gernsback / Clyde J. Fitch Pianorad:  
- 25 independent vacuum-tube LC oscillators  
- True polyphony (one Audion per key)  
- Tube drift, voltage sag, Audion saturation  
- WRNY 1926 broadcast simulation  

## How to Use
1. Install Cmajor runtime / plugin host  
2. Load `Pianorad_Fitch_Edition.cmajor`  
3. Play MIDI (up to 25-note chords)  

Based on historical Radio News articles (Nov/Dec 1926).
